Customer With Phone.lvclass:Handle Messages.vi

This is the main method handling incomming messages to the process. This method should be overridden to add functionality to the process or override functionality implemented by a parent. Any message not handled should be sent to the parent method using the default case.

Each state should be atomic and states should not enqueue new messages from within this method. For cyclic iteration of an action, using a Helper Loop is recommended.
